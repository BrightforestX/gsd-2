# OpenSpec / meta-specs (bootstrap)

This folder is a lightweight placeholder for spec-driven workflows aligned with GSD.

- Use `npx @fission-ai/openspec` or your chosen OpenSpec CLI to scaffold richer artifacts when needed.
- GSD runtime does **not** depend on this directory; it exists so maintainers can grow machine-readable specs without blocking the CLI.

See [docs/contributing-meta-specs.md](../docs/contributing-meta-specs.md) for how this relates to `.gsd/`.
